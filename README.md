# ryanlib - VEX programming library for plug and play autonomous control

## Features
- Motion Profile for fast, consistent linear and turn movements
- Supports S Curve and Trapezoidal motion profiles
- Async control means you can control multiple subsystems at once
- Allows profiled feedforward control for accurate velocity
- Trajectory generation if you are able to generate them

## How To Install
1. Go to releases and download the latest version
2. Open a terminal wherever you downloaded it (try shift + right click in file explorer)
3. run 'pros c fetch ryanlib@VERSION.zip', replacing VERSION with the version
4. run 'pros c apply ryanlib' in your project

Alternatively, the repository includes a pros project with ryanlib already installed, simply clone this repository and the code should be ready to use
1. Go to the top right corner, click code, and copy the link to the repository
2. Go to where you want the project to be, open up a terminal (shift + right click), type 'git clone <link>'
